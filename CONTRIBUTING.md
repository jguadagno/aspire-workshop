# Contributing

Thanks for your interest in improving this workshop! Contributions are welcome.

## How to Contribute

1. **Fork** this repository and create a feature branch.
2. Make your changes — fix a typo, improve a lab, or add new content.
3. **Test** any code changes by running the relevant lab locally.
4. Open a **pull request** with a clear description of your changes.

## Reporting Issues

If you find a bug in a lab or have a suggestion, please [open an issue](../../issues/new) with:

- Which lab or section is affected
- What you expected vs. what happened
- Your environment (OS, .NET version, Docker version)

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to uphold it.
